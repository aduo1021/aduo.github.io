第一炮：学习使用Markdown写文章

前言

    不多说，先废话一波。弄个博客装逼这么久了，还没写点东西，真是失败。找个借口–被项目忙晕了，没空来打理你，以后有空会经常来照顾你的！一直久仰Markdown，一直没闲心来实践下你（懒），今天终于鼓起老底决定来搞定你，以后得时常用你来谢谢心得或笔记的。作为一个接近2年的前端程序猿没有一个自己的博客是有点丢脸，即使没有像张鑫旭大神和阮一峰大神的博客那么有质量，但是记录下自己人生路上的点点滴滴还是有意思的。

Markdown语法

    开始学习写Markdown文章还是记录下语法，虽然简单易学，但以后万一记不得了，至少是翻自己的blog而不是去找度娘了（强行装了一波逼）。废话完结，开始吧。。。

  1. 标题

    一个#表示一级标题，如

    # 一级标题

    ## 二级标题

    ### 三级标题

  2. 列表
    有序列表使用数字接着一个英文句点

    1.天空
    2.大地

    无序列表是使用,+,-中任意一种来表示

    -Red
    +Green
    *Blue

  3. 段落和换行
    在文本末尾打上2个空格即可

  4. 分割线
    一行中用三个以上的星号、减号、底线来建立一个分隔线

  1. 链接
    [文本内容](链接)

  2. 代码块
    建立代码区块,只需要简单地缩进4个空格或是1个制表符就可以
    如果要标记一段行内代码,可以用反引号 ` 把它包起来 用多个反引号来开启和结束代码区段
    hello world

  3. 强调
    两端被一个或_包围的单词会被转换成斜体 两端被两个或_包围的单词会被转换成粗体

    天空
    大地

  4. 转义
    使用反斜杠来插入一些在语法中有其它意义的符号
    \你的内容

    更多详细语法参考

    ##总结
        写了一堆什么乱七八糟的东西哟，基本的语法敲了一遍，二级标题下面是不是自动带分割线的？！感觉写得巨丑，无法直视，晕死。合上盖子睡觉去了，下次再见！
